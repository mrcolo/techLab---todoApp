# techLab Snippets - todoApp

A simple todo app realized with Jquery, Bootstrap and ExpressJS.

### Installing

1. Download the repo on your computer.

```
git clone https://github.com/mrcolo/todoApp.git
```

2. Install the modules

```
npm install
```
